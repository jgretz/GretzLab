Cereal

Cereal is a lightweight serialization framework for iOS. Its uniqueness is that it will serialize and deserialize objects from / to their actual class not raw dictionaries and arrays. This means no more fat fingering the name of a property, you get intellisense and compile time checks.

It does depend on my IOC / DI framework (CoreMeta). the benefit is that due to that all objects that are deserialized will also be injected, subject to type redirection, and all the other IOC / DI goodies.

It is currently written in Objective-C, although it is compatible with Swift. I am currently rewriting it in Swift and doing some upgrading and clean up along the way.

It will serialize and deserialize from the following formats:

  • JSON
  • XML
  • CSV
